# Underworld Deckbuilder 🃏⚔️

A fast, modern, and native Android application designed for building and managing decks for the **Warhammer Underworlds** tabletop game. Built with Vue.js, Vite, and wrapped seamlessly for Android.

[![Boosty](https://shields.io)](ССЫЛКА_НА_ВАШ_BOOSTY)

---

## ⭐ Key Features

*   **📱 100% Offline First:** No internet? No problem. All card databases and your created decks are stored completely locally on your device. Perfect for tournament environments or gaming clubs with poor cellular reception.
*   **⚡ Ultra-Fast UX:** Instant filtering by warbands, card types, and game formats. No annoying web-page reloads or interface lag.
*   **🔗 Advanced Deck Sharing:** Share your tactical setups with friends instantly via lightweight, encoded links. One click is all it takes to import.
*   **🚫 No Annoying Ads:** Enjoy a clean, distraction-free environment dedicated entirely to your strategy experiments.

## 🛠️ Project Setup

If you want to run the project locally or contribute to the development:

```bash
# Install dependencies
npm install

# Compile and Hot-Reload for Development
npm run dev

# Compile and Minify for Production
npm run build
```
